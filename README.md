### 📌 Purpose

- Try out Git commands without fear! 😆
- Learn about **branches, commits, merges, stashes...**
- Make mistakes and learn from them! 🤯🔥

### 🚀 How to Use This Repo

1. **Clone the repo:**
   ```bash
   git clone <repo-link>
   ```
2. **Create and switch to your own branch:**
   ```bash
   git checkout -b my-branch
   ```
3. **Experiment freely!** 😍
